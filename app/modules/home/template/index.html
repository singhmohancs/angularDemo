Welcome to, Employee Management panel

This is an single page application to manage employee with CURD operation. This application is easy to extend with more features.

This application is having following features

  • Listing all employee
  • Easy navigation
  • Add employee(Client side form validations)
  • Edit employee(Client side form validations)
  • Delete employee

AngularJs Modules

  • Angular UI Router
  • Angular Twitter bootstrap
  • Angular LocalStorage
